This section lists and introduces the blogs posted to [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj).

### [Blog 1 - I've Been Using Kiro for 2 Weeks](3.1-Blog1/)
A firsthand experience using Kiro — an AWS AI code editor with a spec-driven workflow. The author covers the workflow, standout features like Steering Files, Hooks, and MCP, plus honest pros and cons after two weeks of daily use. Includes the story of discovering Kiro through a Saturday evening session.

### [Blog 2 - SAM – Deploying AWS Without Clicking Through Console](3.2-Blog2/)
An experience sharing on using AWS SAM (Serverless Application Model) to deploy serverless infrastructure instead of manually clicking through Console. The author covers the workflow with `template.yaml`, `sam build`, `sam deploy`, and the "headaches" encountered like Circular Dependency and lessons from Vibe Coding.

### [Blog 3 - Automating PostgreSQL Audit Log Extraction and Analysis with Amazon S3](3.3-Blog3/)
A solution for automating audit log processing for Amazon RDS PostgreSQL using pgAudit, CloudWatch Logs subscription filters, Lambda, and S3. Logs are parsed into structured CSV format, queryable with Athena, with cost analysis and security considerations.
