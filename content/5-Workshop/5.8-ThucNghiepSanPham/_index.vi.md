### Demo sản phẩm HireFlow

Dưới đây là các hình ảnh demo thực tế của sản phẩm HireFlow - hệ thống tự động hóa tuyển dụng sử dụng AI.

### Lưu ý khi sử dụng

> **Quan trọng:** Không được gửi 2 file cùng lúc. Nếu Lambda phát hiện duplicate (2 file trùng nội dung), file sẽ không xuất hiện trên Dashboard HR.

### Hình ảnh demo

#### So sánh ứng viên

![So sánh ứng viên](/images/5-Workshop/demo1.png)

Giao diện so sánh các ứng viên cùng vị trí, AI phân tích và chấm điểm từng ứng viên dựa trên các tiêu chí.

![So sánh điểm](/images/5-Workshop/demo2.png)

Chi tiết điểm số từng tiêu chí: Experience, Skills, Education, Certifications. AI tự động so sánh và xếp hạng ứng viên.

#### Phân tích ứng viên Security Intern

![Phân tích Security Intern](/images/5-Workshop/demo3.png)

Khi AI phân tích ứng viên Security Intern, hệ thống sẽ:
- **Chỉ ra các điểm còn thiếu:** Các kỹ năng, chứng chỉ hoặc kinh nghiệm mà ứng viên chưa đáp ứng đầy đủ
- **Đánh giá tiềm năng:** Dựa trên thông tin CV, AI đánh giá mức độ phù hợp và tiềm năng phát triển của ứng viên
- **So sánh với yêu cầu JD:** Đối chiếu năng lực ứng viên với Job Description để đưa ra đề xuất
