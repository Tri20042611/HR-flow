### HireFlow Product Demo

Below are actual demo images of HireFlow - an AI-powered recruitment automation system.

### Product Links

- **Web upload CV (candidate):** https://d1jr7i4kzp91u.cloudfront.net/
- **Web dashboard HR:** https://d1gvdj2kjulw2q.cloudfront.net/

### Usage Note

> **Important:** Do not upload 2 files at the same time. If Lambda detects duplicates (2 files with the same content), the file will not appear on the HR Dashboard.

### Demo Images

#### Candidate Comparison

![Candidate Comparison](/images/5-Workshop/demo1.png)

Interface comparing candidates for the same position, AI analyzes and scores each candidate based on criteria.

![Score Comparison](/images/5-Workshop/demo2.png)

Detailed scores for each criteria: Experience, Skills, Education, Certifications. AI automatically compares and ranks candidates.

#### Security Intern Candidate Analysis

![Security Intern Analysis](/images/5-Workshop/demo3.png)

When AI analyzes Security Intern candidates, the system will:
- **Point out missing areas:** Skills, certifications, or experience that the candidate hasn't fully met
- **Evaluate potential:** Based on CV information, AI assesses the suitability and development potential of the candidate
- **Compare with JD requirements:** Match candidate capabilities against the Job Description to provide recommendations
